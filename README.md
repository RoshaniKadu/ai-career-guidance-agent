**AI-Powered Resume Analyzer \& Career Guidance Agent**



**SDG Alignment:**

SDG 8 – Decent Work and Economic Growth



**Project Overview:**



This project is a Streamlit-based AI career guidance application that helps job seekers identify suitable job roles based on their skills, education, and experience.

It aims to reduce skill–job mismatch and improve employment readiness using AI-driven analysis.



**Key Features:**



-Skill-based job role recommendations

-Resume and employment readiness scoring

-Skill gap identification

-Salary insights based on experience level

-Interactive web-based interface



**Tools \& Technologies:**



-Python

-Streamlit

-Rule-based AI logic (prototype)



**AI Workflow:**



**Input:** Skills, education, experience

**Processing:** Skill matching, readiness scoring, gap analysis

**Output:** Job recommendations, missing skills, salary insights



**SDG Impact:**



Supports SDG 8 by promoting decent work opportunities, improving employability, and encouraging targeted upskilling.



**Prototype Status:**



Functional prototype implemented as a Streamlit web application.



**Requirements:**

\- Python 3.x

\- Streamlit





**How to Run:**

**1. Install dependencies**

**pip install streamlit**



**2. Run the application**

**streamlit run app.py**
